# Cómo contribuir a IKI Network

Este proyecto está en Fase Alfa. Las formas 
principales de contribuir hoy son:

1. **Únete como Founding Node** si tienes 
   capacidad de cómputo disponible
2. **Comparte feedback** sobre el Whitepaper 
   en nuestro Discord
3. **Reporta ideas o preguntas** abriendo un 
   Issue en este repositorio

Pronto abriremos contribuciones de código 
conforme el desarrollo avance.
